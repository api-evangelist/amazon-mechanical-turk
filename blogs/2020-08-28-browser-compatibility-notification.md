<p>Microsoft will soon disable Transport Layer Security (TLS) 1.0 and 1.1 by default in Microsoft Internet Explorer and Microsoft Edge browsers. As a result of these upcoming changes, Amazon Mechanical Turk will no longer support legacy versions of Microsoft Internet Explorer and Microsoft Edge browsers. Please update your browser to the latest version to avoid disruption. Workers attempting to reach <a href="http://worker.mturk.com/">worker.mturk.com</a> using legacy browsers that are no longer supported after August 28, 2020 will encounter an error message.<br /> <br />MTurk is broadly supported across modern browsers including but not limited to <a href="https://www.google.com/chrome/">Chrome</a>, <a href="https://www.mozilla.org/en-US/firefox/">Firefox</a> and the new (<a href="https://www.chromium.org/Home">Chromium</a> based) <a href="https://www.microsoft.com/en-us/edge#:~:text=Download%20the%20new%20Microsoft%20Edge,10%20PC%20and%20restart%20it.&amp;text=After%20installation%2C%20these%20terms%20are,Edge%20at%20edge%3A%2F%2Fterms.">Microsoft Edge</a>. Please ensure that you have the latest version installed.<br /> <br />If you have questions or need additional information, please reach out to support at <a href="https://www.mturk.com/contact-us">https://www.mturk.com/contact-us</a>.</p><img alt="" height="1" src="https://medium.com/_/stat?event=post.clientViewed&amp;referrerSource=full_rss&amp;postId=e64b81a1570e" width="1" /><hr /><p><a href="https://blog.mturk.com/browser-compatibility-notification-e64b81a1570e">Browser Compatibility Notification</a> was originally published in <a href="https://blog.mturk.com">Happenings at MTurk</a> on Medium, where people are continuing the conversation by highlighting and responding to this story.</p>
